What is the document object model?

DOM is a kind of programming inerface that helps developers to interacts with webpages using JS. It makes HTML document as a object and helps developers to manipulate HTML and CSS using Javascript. For example, if a developer want to change a heading text, then using DOM, they can easily do it. Whereas, DOM make it as an object and then developer can modify it easily with javascript file.

How do you select an element from the DOM?

You can pick elements based on their ID, class, tag name, or CSS selectors. This allows developers to change things like text, style, or attributes on the page. With these methods, you can create interactive features, such as changing colors, updating content, or hiding and showing parts of the page based on what users do.

What is event delegation in the context of the DOM, and why it is useful?

Event delegation in the DOM is a way to manage events by placing a single event listener on a parent element, instead of adding one to each child element. When a child element triggers an event, it "bubbles up" to the parent, which handles it. This method is helpful because it makes the page faster, especially if you have many similar elements, and it also works well with new elements added later, without needing extra event listeners.
